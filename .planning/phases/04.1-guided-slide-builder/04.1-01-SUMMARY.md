## What Was Built

### Image Utilities (imageUtils.ts)
- `validateImageFile(file)`: Rejects non-PNG/JPG files and files over 6MB with descriptive error messages
- `resizeImageToBase64(file)`: Reads via FileReader, resizes to max 800px via Canvas, strips data URL prefix for Office.js compatibility

### Guided Prompt Builder (promptBuilder.ts)
- `buildGuidedPrompt(brandName, purpose)`: Reuses existing SLIDE_SCHEMA, adds brand-specific focus and analysis purpose context. Does NOT send the image to Cube AI.

### Types and Constants
- `WizardData` interface: brandName, productImageBase64, purpose
- `WizardStep` type: 1 | 2 | 3 | 4
- `BuildState` type: idle | building | built | failed
- `IMAGE_REGION` constant: Same position as CHART_TEXT.CHART for consistent layout

### Extended Slide Renderer (slideRenderer.ts)
- `insertSlide(content, productImageBase64?)`: Optional second parameter renders product image via `ShapeFill.setImage()` on a borderless rectangle shape within the same PowerPoint.run() call
