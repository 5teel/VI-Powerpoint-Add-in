**Step 1: Install Yeoman and generator globally.**
```bash
npm install -g yo generator-office
```

**Step 2: Scaffold the project** using the Yeoman generator with non-interactive CLI flags. Run from the project root directory `C:\Development\Integration\Powerpoint`:
```bash
yo office --projectType taskpane --name "Summit VI" --host powerpoint --ts true --framework react --output .
```
If `yo office` does not accept `--output .` or places files in a subdirectory, move all generated files to the project root. The goal is that `package.json` lives at `C:\Development\Integration\Powerpoint\package.json`.

If `yo office` fails or the flags are not supported in v3.0.2, scaffold interactively by running `yo office` and selecting:
- Project type: "Office Add-in Task Pane project using React framework"
- Script type: TypeScript
- Name: "Summit VI"
- Office host: PowerPoint

**Step 3: Pin React 18 and add Fluent UI v9** (per D-07). After scaffold completes:
```bash
npm install react@18.3.1 react-dom@18.3.1
npm install @fluentui/react-components@9.73.4 @fluentui/react-icons
```
Verify `package.json` shows `react` at `18.3.1`, NOT `19.x`. If React 19 was installed by the scaffold, the above commands will downgrade it.

**Step 4: Enable TypeScript strict mode.** Open `tsconfig.json` and ensure it contains:
```json
{
  "compilerOptions": {
    "strict": true
  }
}
```
If `strict` is already `true`, leave it. If the scaffold set it to `false` or omitted it, add `"strict": true`.

**Step 5: Verify Office.js CDN script tag** in `src/taskpane/taskpane.html`. The file MUST contain this script tag in the `<head>`:
```html
<script type="text/javascript" src="https://appsource.microsoft.com/lib/office/office.js"></script>
```
Do NOT remove or modify this tag. Do NOT install `office-js` via npm.

**Step 6: Update the React entry point** (`src/taskpane/index.tsx`) to use FluentProvider with webLightTheme and Office.onReady:
```typescript
import React from "react";
import { createRoot } from "react-dom/client";
import { FluentProvider, webLightTheme } from "@fluentui/react-components";
import App from "./components/App";

Office.onReady((info) => {
  if (info.host === Office.HostType.PowerPoint) {
    const container = document.getElementById("container");
    if (container) {
      const root = createRoot(container);
      root.render(
        <FluentProvider theme={webLightTheme}>
          <App />
        </FluentProvider>
      );
    }
  }
});
```

**Step 7: Create a minimal App component** at `src/taskpane/components/App.tsx` that renders a simple "Summit VI" text. This is a placeholder -- Plan 02 builds the full UI.
```typescript
import React from "react";

const App: React.FC = () => {
  return <div>Summit VI - Loading...</div>;
};

export default App;
```

**Step 8: Clean up the scaffold's default CSS.** Clear the contents of `src/taskpane/taskpane.css` and replace with a minimal reset:

  margin: 0;

**Step 9: Update .gitignore** to include `.env` (must not be committed), `node_modules/`, `dist/`, and any cert files. The scaffold may already have a `.gitignore`; ensure `.env` is listed.

**Step 10: Verify the project compiles:**
