  <action>
    **numberFormatter.ts** — implements D-07 auto-format:

    Export `formatCellValue(raw: string | number): FormatResult`:
    - Currency: input starts with `$` or is a number → format as `$#,##0` or `$#,##0.00`. Detection: `/^\$[\d,.]+$/` or explicit flag.
    - Percentage: input ends with `%` → keep as-is, mark isNumeric=true. Detection: `/^[\d,.]+%$/`.
    - Large integer: parseable number >= 1000 with no decimals → `toLocaleString("en-US")`. Detection: `Number.isInteger(num) && num >= 1000`.
    - Decimal: parseable number with decimals → keep as-is with `toLocaleString`. Detection: `!isNaN(parseFloat(str))`.
    - Plain text: everything else → `{ display: str, isNumeric: false }`.

    Import `FormatResult` from `./types`.

    **textRenderer.ts** — three exported functions:

    1. `addTitle(shapes: PowerPoint.ShapeCollection, text: string, region: LayoutRegion): PowerPoint.Shape`
       - `shapes.addTextBox(text, { left: region.left, top: region.top, width: region.width, height: region.height })`
       - Font: Calibri 28pt bold, color #0F1330 (COLORS.SUMMIT_NAVY)
       - Returns the shape reference

    2. `addBody(shapes: PowerPoint.ShapeCollection, bullets: string[], region: LayoutRegion): PowerPoint.Shape`
       - Join bullets with `\n`, prefixing each with `\u2022 ` (Unicode bullet per Pitfall 6)
       - `shapes.addTextBox(bulletText, { left, top, width, height })`
       - Font: Calibri 18pt regular, color #333333 (COLORS.BODY_TEXT)
       - `textFrame.leftMargin = 18` (BULLET_INDENT)
       - Returns the shape reference

    3. `addCalloutBox(shapes: PowerPoint.ShapeCollection, text: string, region: LayoutRegion): PowerPoint.Shape`
       - Create main rectangle via `addGeometricShape(PowerPoint.GeometricShapeType.rectangle, { left, top, width, height })`
       - Fill: #EBF0FA (COLORS.CALLOUT_BG)
       - Text: Calibri 22pt bold, color #0F1330 (COLORS.CALLOUT_TEXT)
       - Vertical alignment: middle
       - Margins: left=18, top=12, bottom=12, right=12 (per UI-SPEC)
       - Create accent border: `addGeometricShape(rectangle)` at same left, same top, width=3, same height, fill #0F1330 (COLORS.CALLOUT_BORDER)
       - DO NOT attempt addGroup in this function — grouping requires a sync first (Pitfall 5). The two shapes render correctly without grouping. Grouping will be handled by the slide renderer after sync if needed.
       - Returns the main box shape reference

    Import constants from `./constants` and types from `./types`. All functions are synchronous (they add proxy objects to the batch; caller handles context.sync).
  </action>
